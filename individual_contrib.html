This is the portion of our project where a world map would display and when the user clicked on the information, th country's name among other rellevant information would appear.

  • Selecting Elements:
    • const worldMap = document.getElementById('world-map');: Retrieves the 'world-map' element.
    • const countryLabel = document.getElementById('country-label');: Retrieves the 'country-label' element.
  • Adding Event Listener:
    • worldMap.addEventListener('click', (event) => { ... });: Sets up a click event listener on 'world-map'.
  • Checking Click Coordinates:
    • Retrieves X and Y coordinates of the click event relative to 'world-map'.
    • Defines an array countries with country objects and their coordinates.
  • Iterating through Countries:
    • Iterates through countries array.
    • Checks if click coordinates fall within the specified range for each country.

      • Code 

              // Inline JavaScript for interactivity
        const worldMap = document.getElementById('world-map');
        const countryLabel = document.getElementById('country-label');

        worldMap.addEventListener('click', (event) => {
            const x = event.offsetX;
            const y = event.offsetY;

            // You can define coordinates and country names here.
            const countries = [
                { name: 'USA', x1: 100, x2: 200, y1: 100, y2: 200 },
                { name: 'Canada', x1: 300, x2: 400, y1: 150, y2: 250 },
                // Add more countries and their coordinates as needed.
            ];

            for (const country of countries) {
                if (x >= country.x1 && x <= country.x2 && y >= country.y1 && y <= country.y2) {
                    countryLabel.textContent = country.name;
                    countryLabel.style.left = `${x}px`;
                    countryLabel.style.top = `${y}px`;
                    countryLabel.style.display = 'block';
                    break;
                } else {
                    countryLabel.style.display = 'none';
                }
            }
        });
