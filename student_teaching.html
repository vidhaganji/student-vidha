       "# Popcorn Hack (API)\n",
       "Problem: You want to create a Python program that fetches a random fact from the Cat Facts API and displays it. Write a Python script that retrieves a random cat fact from the API and prints it to the console.\n",
       "\n",
       "Assumptions/Hints:\n",
       "You can make an HTTP GET request to the API using the requests library.\n",
       "The API endpoint for fetching a random cat fact is: https://meowfacts.herokuapp.com/1.\n",
       "After getting the response, you can extract the fact from the JSON response.\n"
      ]
     },
     {
      "cell_type": "code",
      "execution_count": null,
      "metadata": {},
      "outputs": [],
      "source": [
       "import requests\n",
       "\n",
       "# Define the API endpoint for cat facts\n",
       "API_URL = 'https://meowfacts.herokuapp.com/1'\n",
       "\n",
       "# Make an HTTP GET request to the API\n",
       "response = requests.get(API_URL)\n",
       "\n",
       "# Check if the request was successful (status code 200)\n",
       "if response.status_code == 200:\n",
       "    # Parse the JSON response to get the cat fact\n",
       "    cat_fact = response.json()['data'][0]\n",
       "    \n",
       "    # Print the cat fact\n",
       "    print(\"Random Cat Fact: \" + cat_fact)\n",
       "else:\n",
       "    print(\"Failed to retrieve a cat fact. Please try again later.\")\n"
